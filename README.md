deafult.zipをDL後、中身を作業ディレクトリの中にコピーしてPCにnode.jsとgulp(global)をinstallしてください
一応node.jsをインストールしたあとPCをRebootすること

コンソールから作業directoryに移動して

GulpをGlobalにinstallします
>npm install gulp -g

そのあと
> $ npm install

ちなみにgulp起動コマンドは
> $ npx gulp start
