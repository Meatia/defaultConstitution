# Gulp StartUp File
### HTML5/CSS3/FLOCSS

自分用に作ったFLOCCSベースのディレクトリ構造

deafult.zipをDL後、中身を作業ディレクトリの中にコピーしてPCにnode.jsとgulp(global)をinstallしてください
一応node.jsをインストールしたあとPCをRebootすること

コンソールから作業directoryに移動後

GulpをGlobalにinstall
>npm install gulp -g
> $ npm install
で問題なければ終わり。

ちなみにgulp起動コマンドは
> $ npx gulp start

超簡単なものしか入っていないのであとはお好みでどうぞ
